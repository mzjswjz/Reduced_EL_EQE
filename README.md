# Reduced_EL_EQE

Analysis tools for reduced electroluminescence (EL) and external quantum efficiency (EQE) measurements.

## Description

This repository contains Jupyter notebooks for analyzing reduced electroluminescence (EL) and external quantum efficiency (EQE) data. These measurements are commonly used in optoelectronic device characterization, particularly for solar cells and LEDs, to understand device performance and identify voltage losses.

## Requirements

- Python 3.x
- Jupyter Notebook
- NumPy
- SciPy
- Matplotlib
- Pandas

## Usage

1. Clone the repository
2. Install dependencies: `pip install numpy scipy matplotlib pandas jupyter`
3. Open the notebooks in Jupyter: `jupyter notebook`
4. Run the analysis notebooks to process EL and EQE data

## Applications

- Solar cell characterization
- LED performance analysis
- Voltage loss analysis in optoelectronic devices
- Device efficiency calculations
